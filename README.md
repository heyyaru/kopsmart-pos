🛒 KopSmart - Point of Sales (POS) System
KopSmart adalah sistem aplikasi kasir (Point of Sales) modern berbasis web yang dirancang untuk mengelola transaksi penjualan, memantau stok barang, hingga mencatat utang pelanggan secara cepat dan akurat.

✨ Fitur Utama
KopSmart hadir dengan fitur-fitur unggulan untuk mendukung efisiensi operasional:

Dashboard Interaktif: Visualisasi ringkasan omzet, total transaksi, dan notifikasi stok kritis.

Manajemen Kasir (POS): Antarmuka transaksi yang responsif dan intuitif.

Manajemen Produk: Pengelolaan kategori, satuan, serta inventaris barang secara terpusat.

Stock Opname: Fitur stock adjustment untuk sinkronisasi stok fisik dan data sistem.

Manajemen Utang: Pencatatan daftar pelanggan, riwayat piutang, dan status pembayaran.

Laporan & Analitik: Rekapitulasi penjualan serta ekspor riwayat transaksi.

Keamanan & Pengguna: Manajemen hak akses, profil, dan fitur pemulihan kata sandi via email.

💻 Teknologi yang Digunakan
Aplikasi ini dibangun menggunakan tech stack terkini:

Framework: Laravel 11

Frontend: Laravel Livewire & Blade Components

Database: MySQL

Styling: Tailwind CSS
